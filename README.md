# 🍩 Proyecto Three.js FPS

Proyecto desarrollado con :contentReference[oaicite:0]{index=0} utilizando físicas básicas, colisiones Octree y un entorno 3D interactivo estilo FPS.

El jugador puede:

- Moverse con teclado
- Saltar
- Mirar alrededor con el mouse
- Lanzar donas 🍩 con físicas

---

# 📸 Características

✅ Escenario 3D interactivo  
✅ Movimiento tipo FPS  
✅ Sistema de colisiones Octree  
✅ Donas físicas con sprinkles  
✅ Sombras dinámicas  
✅ Navbar y footer personalizados  
✅ Interfaz moderna

---

# 🎮 Controles

| Tecla | Acción |
|---|---|
| W A S D | Movimiento |
| SPACE | Saltar |
| Click sostenido | Mover cámara |
| Soltar click | Lanzar dona 🍩 |
